					===== Для крыльевого БАНО =====

Для редактирования проекта для крыльевого БАНО необходимо:

1. Открыть файл Inc/main.h и закоментировать строку "75" -- "//#define BANO_TALL		// Расскоментировать для хвостового БАНО"

2. На главном меню выбрать "Project" -> "Options for Target 'BanoStm'", далее открыть вкладку "Output" в поле ввода "Name of Executable" написать "BanoWing_v.1.0" (версия ПО) и поставить галочку "Create HEX File"


					===== Для хвостового БАНО =====

Для редактирования проекта для хвостового БАНО необходимо:

1. Открыть файл Inc/main.h и раскоментировать строку "75" -- "#define BANO_TALL		// Расскоментировать для хвостового БАНО"

2. На главном меню выбрать "Project" -> "Options for Target 'BanoStm'", далее открыть вкладку "Output" в поле ввода "Name of Executable" написать "BanoTail_v.1.0" (версия ПО) и поставить галочку "Create HEX File"
